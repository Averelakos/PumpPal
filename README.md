# Pump Pal Documentation
## **1 Dev action**
### 1.1 Run Project Locally
To run the backend first you must build the project. To do that you must run the command in the terminal:`dotnet restore` then `dotnet build` when those actions are complete to run the solution write in the terminal `dotnet run --project ./src/PumpPal.API/PumpPal.API.csproj`.

To run the frontend first you must downloade all the necesery dependencies. To fix that run in the terminal `npm install`. When that is complete then run in the terminal `ng -s` or `ng -s o`.
